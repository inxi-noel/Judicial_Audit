# 祥源账套自动化

这个项目用于批量进入鼎信诺账套树中 `2024.01-2024.12` 二级节点下的四级单位，进入主界面后导出 `7-12` 月的总分类账，并支持断点续跑。

## 目录

- `main.py`
  - 主入口。负责重建进度、扫描账套树、进入项目、按月导出、切回项目树。
- `modules/_shared`
  - 公共配置、窗口重试、日志、导出进度、UI 基础工具。
- `modules/tree_handler`
  - 账套树扫描、进入项目、切换项目并重置树状态。
- `modules/general_ledger`
  - 主界面内的总账明细账多年入口、月份筛选、全部展开、导出 Excel。
- `runtime/`
  - 运行时目录。默认写入日志和导出进度 CSV。

## 断点续跑

- 启动时会扫描默认导出目录 `D:\DXN审计系统安装包_V1.0.4\person`。
- 匹配到 `总分类账-单位名(2024年1月 -2024年X月).xlsx` 的文件，会回填到 `runtime/鼎信诺祥源总分类账日志/export_progress.csv`。
- 后续会按 `单位名 + source_period_key(2024.01-2024.12) + 月份(7-12)` 判断是否跳过。

## 当前假设

- 切换项目的树 UI 与之前的账套项目一致。
- 二级节点文本能识别成 `2024年1月—2024年12月` 这一类期间格式。
- 主界面上的关键控件位置与原型脚本一致：
  - 总账明细账多年入口 `t_cwsj_2_5`
  - 月份下拉框、查询按钮
  - 全部展开、是(Y)、有数据的展开无数据隐藏、确定
  - 导出到 EXCEL

如果其中任一控件的文本、坐标或窗口层级发生变化，需要先更新 `modules/_shared/config.py` 中的常量。
